Installation

In your project:

  1. Open a package manager window. (menu: Window > Package Manager).
  2. Click the plus button and select "Add package from git URL...".
  3. Paste the link: https://github.com/kempnymaciej/alchemy-core.git?path=/package#v1.0.0. (Replace 1.0.0 with version you would like to download.)
  4. Add AlchemyBow.Core reference to your assembly definition.

Alternativlely, you can clone the package repository.
